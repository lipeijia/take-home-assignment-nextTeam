## 📂 項目結構

### ReactLikeDislikeTypescript
Like/Dislike 按鈕組件，具有狀態管理功能
- [查看項目](./ReactLikeDislikeTypescript)

### RunningClock
可配置的倒數計時器，支援啟動、暫停、重置功能
- [查看項目](./RunningClock)

###  IsCycleCheck - 圖論算法
檢查有向圖是否形成簡單循環的算法實現
- [查看項目](./IsCycleCheck)

## 🚀 使用方式
每個項目都有獨立的 README 文件，包含詳細的安裝和運行說明。


# 安裝所有依賴
npm run install:all

# 運行 React Like/Dislike 項目
npm run dev:react-like

# 運行計時器項目  
npm run dev:clock

# 測試循環檢測算法
npm run test:cycle

# 訪問 
http://localhost:3000
